## Key Insights

### What Makes This Different from OzBargain

| Aspect | OzBargain | UAE Bargains |
|--------|-----------|---------------|
| **Market** | Australia | UAE (AED) |
| **Currency** | AUD | UAE Dirham |
| **Culture** | Australian shopping patterns | UAE consumer behavior |
| **Retailers** | Australian brands | UAE-specific retailers |
| **Social** | Facebook groups | WhatsApp + fragmented discovery |
| **Mobile** | High smartphone penetration | 60-70% mobile users |

### Voting System (OzBargain Reference)

- **Upvote/Downvote** on BOTH deals AND comments
- **Negative vote revocation:** Requires 9+ dissenting opinions
- **Rule:** 5x comments = 5x more likely to have neg vote revoked
- **Purpose:** Prevent spam, reward quality content, balance discussion

## Tech Stack Confirmed

- **Frontend:** React + Next.js
- **Backend:** Supabase (for user auth) OR Firebase (to be confirmed)
- **Database:** PostgreSQL (deal relationships)
- **Hosting:** Vercel
- **UI:** Tailwind CSS or shadcn/ui
